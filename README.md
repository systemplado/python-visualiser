# Python-Visualiser

[Link to original repo](https://github.com/n00bsys0p/python-visualiser)

## Setup
```
    sudo apt-get install python-numpy
    sudo apt-get install python-matplotlib
```
- [pydub](https://github.com/jiaaro/pydub#installation) should be installed

## Running the application
- use `importer.py` to import a .wav stereo file to a .wav mono file
- run `make.sh` with the arguments `input_sound output_video`
